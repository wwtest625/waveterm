Wave Terminal 是一款开源 AI 原生终端，支持 macOS、Linux、Windows 三大平台，集成对话式 AI 助手、持久化 SSH 连接、灵活的面板布局，让开发者的工作流更加高效且智能。

核心特性

Wave 将 AI 能力深度融入终端，提供前所未有的智能开发体验。

  • Wave AI 助手 — 上下文感知的终端伴侣，可读取终端输出、分析文件、执行操作
  • 支持 OpenAI、Claude、Gemini、Azure、Ollama 等多模型
  • 持久化 SSH — 网络中断后可自动重连，会话不丢失
  • 内置文件编辑器，支持语法高亮和远程文件编辑
  • 丰富的文件预览系统（Markdown、图片、视频、PDF、CSV）

技术架构

基于现代化技术栈构建，兼顾性能与开发效率。

  • 前端 — Electron + React 19 + TypeScript + TailwindCSS
  • 后端 — Go 语言实现核心逻辑
  • 终端模拟 — xterm.js
  • 代码编辑 — Monaco Editor
  • 包管理 — npm workspaces
