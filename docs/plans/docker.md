# Docker 侧边栏产品文档

## 概述

在 WaveTerm 侧边栏添加 Docker 面板，无需离开终端即可管理容器和镜像。

## 入口

侧边栏 右侧新增 Docker 按钮，点击展开右侧可折叠面板。

## 功能

### 容器管理

| 功能 | 说明 |
|------|------|
| 列表 | 显示容器名、镜像、状态、端口 |
| 状态 | 🟢 Running / 🔴 Stopped / 🟡 Paused |
| 操作 | Start / Stop / Restart / Delete |
| 日志 | 点击查看实时日志 |
| 终端 | 双击或右键 Exec 进入容器 |

### 镜像管理

| 功能 | 说明 |
|------|------|
| 列表 | 镜像名、Tag、大小 |
| 拉取 | 输入镜像名拉取 |
| 删除 | 清理镜像 |

## 交互

- 单击容器 → 展开详情
- 右键 → 上下文菜单
- 双击 → 新 Tab 打开容器终端


## 优先级

1. 容器列表 + 基本操作
2. 镜像列表
3. 日志查看
4. Exec 终端
